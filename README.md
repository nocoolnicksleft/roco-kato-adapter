# Roco to Kato track adapters

These parts are made to connect vintage Roco N type track to Kato Unitrack.

It is accomplished by a heightened bed containing a cutout to avoid the track sliding,
and a simplified version of the Kato Unijoins.


If doing further development, I'd be pleased if you sent them to me, or make a branch at Github:

# Customizing

Several Roco track pieces are prefined in the SCAD model.


# Building

## Hints

Glueing the track to the bed is recommended in installations. Be sure to do so while the piece is connected to Unitrack on all sides, to avoid minor variations.

I have used PETG for higher durability, but they work as PLA just as well.

## Print Settings

| Parameter        | Value              |
|------------------|--------------------|
| Layer Height     | 0.1 mm             |
| Infill           | 20%                |
| Infill Pattern   | Gyroid             |
| Supports         | No                 |
| Material         | PLA or PETG        |
| Walls/Perimeters | 3                  |
| Printer          | Prusa MK3          |
| Slicer           | PrusaSlicer 2.x    |
